Desafío 01 | Calculadora de BPM a MS█

Este programa corresponde al Desafío Nro01 del curso de Javascript.
El mismo se trata de una calculadora que convierte BPM (Beats Per Minute) a milisegundos.

Para esto mismo, comenzaremos seleccionando de nuestro input un valor expresado en BPM. Este mismo ya estará seteado en 120BPM por default.

A partir de este dato, nuestro programa podrá convertir ese dato a una variable llamada "bpm".

Además de esto, para obtener nuestro resultado podremos variar entre las dos figuras más conocidas con las cuales vamos a realizar nuestras subdiviciones
